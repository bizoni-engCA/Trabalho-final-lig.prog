## 1. Resumo
Este projeto implementa e compara três abordagens para armazenamento e consulta de dados de sensores de temperatura industriais: uma **Lista Encadeada Ordenada** (Insertion Sort), uma **Árvore Binária de Busca Rubro-Negra** e uma **Heap**. O objetivo é analisar a escalabilidade de cada método conforme o volume de dados cresce.

## 2. Descrição do Problema e Solução
O sistema legado da empresa, baseado em listas simples, apresenta lentidão em consultas com alto volume de sensores. Para resolver isso, propomos a comparação entre manter a lista ordenada na inserção versus utilizar uma estrutura hierárquica (Árvore) para otimizar as buscas e inserções.
